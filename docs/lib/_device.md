# device对象
device对象是获取设备信息和基于设备操作的属性

## isMobile
`isMobile`判断是否是移动端
```js
/**
 * 判断是否是移动端
 * isMobile
 * @return { Boolean } 返回是否是移动端的布尔值
 */
```
##### `Demo`:
```js
Dutils.device.isMobile() // false
```

## isIOS
`isIOS`判断设备是否是IOS系统
```js
/**
 * 判断是否是IOS操作系统
 * @return { Boolean } 返回是否是IOS的布尔值
 */
```
##### `Demo`:
```js
Dutils.device.isIOS() // false
```
